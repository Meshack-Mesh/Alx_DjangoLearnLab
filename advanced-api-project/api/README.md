### Filtering
Use query parameters to filter results:
- `/api/books/?title=Example`
- `/api/books/?publication_year=2005`
- `/api/books/?author=1`

### Searching
Search across specific fields:
- `/api/books/?search=tolkien`
- `/api/books/?search=magic`

### Ordering
Sort results:
- `/api/books/?ordering=title`
- `/api/books/?ordering=-publication_year`
